题目描述

近些年来，我国防沙治沙取得显著成果。某沙漠新种植 N N N棵胡杨（编号 1 − N 1-N 1−N），排成一排。

一个月后，有 M M M棵胡杨未能成活。

现可补种胡杨 K K K棵，请问如何补种（只能补种，不能新种），可以得到最多的连续胡杨树？

输入描述

N N N 总种植数量， 1 ≤ N ≤ 100000 1 ≤ N ≤ 100000 1≤N≤100000

M M M 未成活胡杨数量， M M M 个空格分隔的数，按编号从小到大排列， 1 ≤ M ≤ N 1 ≤ M ≤ N 1≤M≤N

K K K最多可以补种的数量， 0 ≤ K ≤ M 0 ≤ K ≤ M 0≤K≤M

输出描述

最多的连续胡杨棵树

样例1

输入

5
2
2 4
1

输出

3

说明

补种到 2 2 2或 4 4 4结果一样，最多的连续胡杨棵树都是 3 3 3。

样例2

输入

10
3
2 4 7
1

输出

6

说明

补种第 7 7 7棵树，最多连续胡杨树棵数位 6 （ 5 ， 6 ， 7 ， 8 ， 9 ， 10 ） 6（5，6，7，8，9，10） 6（5，6，7，8，9，10）

题目解析

思维

结论最优肯定是补中连续的坏树，补中之后，假设0坐标和n+1抖有一颗坏树，答案是两个坏树之间距离-1的最大值，枚举所有可能的补种位置。对于每一组连续的 k 棵坏树，计算它们之间的距离(补种第x颗到第k+x-1颗此时的答案就是(第k+x颗坏树的坐标减去第x-1颗位置-1)，求出最大值即可。

代码如下

cpp

#include <bits/stdc++.h>
using namespace std;
#define int long long
#define N 100005
int a[N];
int n,m;
signed main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	cin>>n;
	cin>>m;
	for(int i=1;i<=m;i++){
		cin>>a[i];
	}
	a[0]=0,a[m+1]=n+1;
	int k;
	cin>>k;
	int ans=k;
	if(k>=m) ans=n;
	for(int i=1;i+k-1<=m;i++){
		int l=a[i-1]+1,r=a[i+k]-1;
		ans=max(ans,r-l+1);
	}
	cout<<ans<<'\n';
	return 0;
}

python

def main():
    n = int(input())
    m = int(input())
    a = list(map(int, input().split()))
    
    a = [0] + a + [n + 1]
    k = int(input())
    ans = k
    
    if k >= m:
        ans = n
    else:
        for i in range(1, m - k + 2):
            l = a[i - 1] + 1
            r = a[i + k] - 1
            ans = max(ans, r - l + 1)
    
    print(ans)

if __name__ == "__main__":
    main()

java

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        long n = scanner.nextLong();
        long m = scanner.nextLong();
        long[] a = new long[(int) m + 2];
        
        for (int i = 1; i <= m; i++) {
            a[i] = scanner.nextLong();
        }
        
        a[0] = 0;
        a[(int) (m + 1)] = n + 1;
        long k = scanner.nextLong();
        long ans = k;

        if (k >= m) {
            ans = n;
        } else {
            for (int i = 1; i + (int) k - 1 <= m; i++) {
                long l = a[i - 1] + 1;
                long r = a[i + (int) k] - 1;
                ans = Math.max(ans, r - l + 1);
            }
        }
        
        System.out.println(ans);
    }
}
